# Gank.io-Rn
React-Natvie 写Gank.io的demo

简单的实现 技术要点
与原生UI的通信，网络的简单封装，listview，和flex布局的掌握

缺陷：ListView 的的刷新机制有问题，图片缩放用的是PhotoView但是没有居中，瀑布流的效果问题

下个阶段要实现的
界面的优化，缓存，Rn中的ListView 的封装（刷新和加载）

界面模仿：https://github.com/CaMnter/EasyGank
